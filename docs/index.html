작동 원리

훅 → 배너 → 응답
전 과정이 자동

Claude Code 훅이 HTTP로 앱에 이벤트를 전달하면, 앱이 배너를 띄우고 응답을 long-poll로 대기합니다.

architecture
┌─ Claude Code (terminal) ──────────────────────────────────────────┐ │ 훅 이벤트 발생 → notify-hook.sh 실행 (stdin: JSON) │ ↓ HTTP POST /event │ ClaudeNotifier app (localhost:47823) │ ↓ NSPanel 배너 표시 │ 사용자가 Yes / Session / No 클릭 │ ↓ HTTP GET /response/{id} (long-poll) │ notify-hook.sh → permissionDecision → Claude Code 허용/차단 └───────────────────────────────────────────────────────────────────┘
01

터미널 포커스 감지

Claude Code가 실행 중인 탭·tmux 창이 전면이면 배너를 생략하고 네이티브 UI를 사용합니다.

02

배너 표시

다른 앱·탭에 있으면 화면 우상단에 플로팅 배너를 쌓아 표시합니다. 소리 없음.

03

응답 전달

버튼 클릭 → HTTP long-poll 응답 → Claude Code가 즉시 결정을 받아 계속 진행합니다.

04

Session 캐시

"Session" 허용 시 해당 세션 동안 동일 도구는 배너 없이 자동 허용됩니다.

05

Fail-open

앱이 실행 중이 아니거나 타임아웃 시 Claude Code 기본 프롬프트로 폴백합니다.

설치

한 줄로 설치

Swift/Xcode 없이 바이너리를 바로 받아 설치합니다.

Terminal
$ curl -fsSL https://raw.githubusercontent.com/uri010/claude-notifier/main/scripts/install.sh | bash
Terminal — 모든 도구 게이트
$ curl -fsSL https://raw.githubusercontent.com/uri010/claude-notifier/main/scripts/install.sh | bash -s -- --all
Terminal — 소스 빌드 (Swift 필요)
$ git clone https://github.com/uri010/claude-notifier.git
$ cd claude-notifier && ./scripts/install-launchagent.sh
$ ./scripts/install-hooks.sh

업데이트는 동일한 curl 명령을 재실행하면 됩니다.  ·  제거: curl -fsSL …/uninstall.sh | bash

01

바이너리 다운로드

GitHub Release에서 유니버설 바이너리(arm64 + x86_64)를 ~/.local/share/claude-notifier/에 설치합니다.

02

LaunchAgent 등록

macOS 로그인 시 자동으로 앱이 시작되도록 LaunchAgent를 등록합니다.

03

훅 등록

~/.claude/settings.json에 PreToolUse·Stop 훅을 자동으로 추가합니다.

04

재시작

Claude Code 세션을 재시작하면 훅이 활성화됩니다. 완료!

설정

환경 변수로 제어

별도 설정 파일 없이 환경 변수만으로 동작을 조정할 수 있습니다.

변수 기본값 설명
CLAUDE_NOTIFIER_PORT 47823 HTTP 서버 포트. 다른 서비스와 충돌 시 변경합니다.
CLAUDE_NOTIFIER_WAIT 600 PreToolUse 응답 대기 시간(초). 이 시간 안에 버튼을 누르지 않으면 fail-open 처리됩니다.
HTTP API (localhost:47823)
메서드경로설명
GET/health앱 상태 확인 → {status, pending, version}
POST/event권한 요청 이벤트 등록 → {id} 반환
GET/response/{id}결정 long-poll (timeout 파라미터 지원)
POST/notify알림 배너 표시 (fire-and-forget)
POST/clear모든 배너 제거
